# Assignment 1 - Circular Buffer

## شو هو المخزن الدائري؟
هو مصفوفة بحجم ثابت تشتغل بشكل دائري،
لما توصل للنهاية ترجع للبداية تلقائياً.

## شو عملت؟
خزّنت اسمي "aya" كـ CE-ESY داخل مخزن
حجمه 2، وهو أصغر من عدد حروف الاسم (3)
عشان يظهر الـ Overflow.

## الدوال:
- init() : تهيئة المخزن
- enqueue() : إضافة عنصر
- dequeue() : قراءة عنصر

## النتيجة:
- CE-ESY1 و CE-ESY2 اتخزنوا بنجاح
- CE-ESY3 طلع Overflow لأن المخزن ممتلئ
- بعد القراءة طلع Underflow لأن المخزن فرغ
